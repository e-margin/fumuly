## 技術スタック

- Next.js（App Router） + Tailwind CSS + shadcn/ui
- Supabase（PostgreSQL + Auth）
- Claude API（Vision解析 + チャット）
- Vercel Pro（ホスティング + Cron Jobs）
- PWA対応（Service Worker + Push通知）

## 主な機能

- 書類スキャン（カメラ / 画像選択 → Claude Vision解析）
- AI チャット相談（書類コンテキスト付き）
- リマインダー通知（Push通知 + Cron自動チェック）
- アカウント管理（表示名・メール・パスワード変更・削除）
- LP（ヒーロー・3ステップモック・CTA）
- PWA（ホーム画面追加・オフライン対応・更新検知）
